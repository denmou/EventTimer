# Don't Starve Mod
## EventTimer
![icon](https://github.com/denmou20/EventTimer/raw/master/preview.png)  
显示事件计时器、可摘取作物生长状态、剩余燃料值  

可配置项：各事件开关、刷新间隔、开始告警时间、文本模式  

默认显示文本为英文、根据游戏显示语言适配中文文本
